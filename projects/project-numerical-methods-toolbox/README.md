# Project: Numerical Methods Toolbox

Collection of computational linear algebra and nonlinear optimization utilities used by the course exercises.

## Includes

- Matrix factorization helpers (`decomp.py`, `models.py`)
- Least-squares solvers and iterative methods
- Newton and Gauss-Newton variants
- Error/conditioning metrics (`metrics.py`)

## Run

```bash
python3 sales.py
python3 satelite.py
python3 tests.py
python3 -m unittest discover -s . -p "test_*.py"
```
