# Project: Quadratic Norm Visualization

Small scripts to visualize ellipsoidal level sets induced by quadratic norms of the form:

\|x\|_A = sqrt(x^T A x)

## Files

- `normas.py`: plots transformed norm contours.
- `norma2.py`: adds eigenvector overlays for geometric interpretation.

## Run

```bash
python3 normas.py
python3 norma2.py
```
