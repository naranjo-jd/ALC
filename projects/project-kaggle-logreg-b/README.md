# Project: Kaggle Logistic Regression B

Extended logistic regression workflow with k-fold validation, F1 scoring, and feature transform search.

## Files

- `clean.py`: preprocessing, normalization, bias, fold splitting
- `model.py`: training, scoring, transform evaluation
- `test.py`: k-fold evaluation script
- `predict.py`: prediction export

## Run

```bash
python3 test.py
python3 predict.py
```
